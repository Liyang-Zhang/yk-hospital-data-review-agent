# PGT-A 指标计算说明（2026-07-01）

## 目的

本文档记录当前 `PGT-A` 业务审核快照的指标实现口径，供开发、医学部和后续接口替换时对齐使用。

当前实现依据：

- 指标需求来源：[PGTA需要统计的指标-20260701.xlsx](/home/zhangly/repo/github/yk-hospital-data-review-agent/docs/PGTA需要统计的指标-20260701.xlsx)
- 数据来源：[PGTA数据统计输出-2025年.xlsx](/home/zhangly/repo/github/yk-hospital-data-review-agent/docs/PGTA数据统计输出-2025年.xlsx)
  - 原始明细 sheet：`2025年-数据`
  - 对照汇总 sheet：`2025年-数据统计`

## 当前统一规则

### 1. 基础记录范围

- 当前所有 `PGT-A` 统计默认使用 `2025年-数据` sheet。
- 当前记录排除规则：
  - `样本类型` 中包含 `对照`
  - `样本类型` 中包含 `极体`
- 除上述质控品/极体外，其余记录都进入当前统计分母。

### 2. 时间口径

- `按年 / 按月 / 按季度` 统计：
  - 优先使用 `月份`
  - 原因：该列与业务整理的 `2025年-数据统计` sheet 月度汇总能直接对齐
- `按日` 统计：
  - 使用 `报告审核时间`

### 3. 主报告结果与附加发现的区分

- `CNV检测结果`
  - 作为主报告层面的 PGT-A 检测结果
- `提示CNV`
  - 作为主报告之外、附件层面的额外发现提示
- `结果解释`
  - 当前用作主报告结果分类：
    - `未见异常`
    - `异常`
    - `嵌合异常`
    - `质控不合格`
- `结果说明`
  - 用于进一步拆解 `提示CNV` 里的综合征、嵌合比例、特殊异常等细项

## 当前已接入执行的指标族

当前代码已围绕以下 6 个执行主题实现一轮：

1. `PGT-A 送检量`
2. `PGT-A 整倍体率`
3. `PGT-A 年龄分层整倍体率`
4. `PGT-A 检测与质控总览`
5. `PGT-A 嵌合与异常结构`
6. `PGT-A 周期结局总览`
7. `PGT-A 特殊 CNV 提示总览`

## 字段与公式说明

### 检测周期数

- 使用列：
  - `送检单编号`
- 计算方式：
  - 去除 `样本类型` 中明显为质控品和极体的记录后
  - 按 `送检单编号` 去重计数

### 检测胚胎数

- 使用列：
  - `样本编号`
  - `样本类型`
- 计算方式：
  - 去除 `样本类型` 中明显为质控品和极体的记录后
  - 剩余记录行数即为检测胚胎数

### 平均囊胚数

- 使用列：
  - `送检单编号`
  - `样本编号`
- 计算方式：
  - `检测胚胎数 / 检测周期数`

### 检测成功率

- 使用列：
  - `CNV检测结果`
  - `结果解释`
- 当前实现：
  - 有主报告 CNV 结果的记录数 / 检测胚胎数
  - 当前代码判断为：
    - `CNV检测结果` 非空
    - 且不为 `/`
    - 且不为 `N/A`
- 备注：
  - 该口径与 `2025年-数据统计` sheet 中的 `检测成功率` 可对齐

### 扩增成功率

- 使用列：
  - 当前尚未稳定确认
- 当前状态：
  - 已识别为医学部需要的指标
  - 但当前原始快照里缺少清晰可回溯的统一阈值说明
  - 本轮未在执行结果中给出正式数值
- 待确认点：
  - `ChromInst` 与 `ChromSwift` 的分母定义
  - 使用 `检测浓度（WGA扩增浓度）` 还是 `建库文库浓度（酶切打断文库浓度）`
  - 成功阈值分别是多少

### 胚胎整倍体率

- 使用列：
  - `结果解释`
- 计算方式：
  - `结果解释 == 未见异常` 的胚胎数 / 检测胚胎数
- 当前实现说明：
  - 医学部需求表标题写了“去除质控不合格的”
  - 但同一表中的公式写的是 `总整倍体胚胎数 / 总检测胚胎数`
  - 当前代码按公式执行，并与 `2025年-数据统计` 的月度汇总对齐

### 胚胎嵌合率（仅嵌合）

- 使用列：
  - `结果解释`
- 计算方式：
  - `结果解释 == 嵌合异常` 的胚胎数 / 检测胚胎数

### 胚胎异常率

- 使用列：
  - `结果解释`
- 计算方式：
  - `结果解释 == 异常` 的胚胎数 / 检测胚胎数
- 备注：
  - 当前不把 `嵌合异常` 并入该分子

### 胚胎异倍体率

- 使用列：
  - `异倍体结果（人工处理）`
- 当前实现方式：
  - 字段值中包含以下任一关键词即计入分子：
    - `三倍体`
    - `UPD`
    - `ROH`
  - 公式：
    - `含三倍体/UPD/ROH 字样的胚胎数 / 检测胚胎数`
- 备注：
  - 该口径是按医学部需求表实现
  - 它比 `2025年-数据统计` sheet 中“异倍体结果占比”的老口径更窄

### 意外发现率

- 使用列：
  - `意外发现（人工处理）`
- 计算方式：
  - `意外发现（人工处理） == 有` 的胚胎数 / 检测胚胎数

## 特殊 CNV 提示类指标

### 1Mb~4Mb 的缺失/重复提示胚胎数数量/率

- 使用列：
  - `提示CNV`
  - `结果说明`
- 当前实现规则：
  - `提示CNV` 中存在 `~X.XXMb`
  - 且大小满足 `1 <= Mb < 4`
  - 且 `结果说明` 中包含 `包含已知综合征`
  - 且不包含 `无已知综合征`
- 当前输出：
  - 已在 `PGT-A 特殊 CNV 提示总览` 中输出数量
- 当前未单独输出比率：
  - 如需比率，可按 `数量 / 检测胚胎数`

### 4Mb~10Mb 且 ≥50% 的嵌合异常胚胎数量

- 使用列：
  - `结果解释`
  - `提示CNV`
  - `结果说明`
- 当前实现规则：
  - `结果解释 == 嵌合异常`
  - `提示CNV` 中大小满足 `4 <= Mb < 10`
  - 且嵌合比例 `>= 50%`
  - 且 `结果说明` 中满足“包含已知综合征”且不为“无已知综合征”

### ≥10Mb 且 20%~30% 的嵌合异常胚胎数量

- 使用列：
  - `结果解释`
  - `提示CNV`
- 当前实现规则：
  - `结果解释 == 嵌合异常`
  - `提示CNV` 中大小满足 `>= 10Mb`
  - 且嵌合比例在 `20%~30%` 之间
- 备注：
  - 按医学部需求表，本项当前不要求判断是否有综合征

### 拟常染色体区域缺失胚胎数量

- 使用列：
  - `提示CNV`
  - `结果说明`
- 当前实现规则：
  - `提示CNV` 中包含 `del(X)` 且包含 `p22.33`
  - 或 `结果说明` 中出现 `X染色体p22.33` 且出现 `缺失`
- 备注：
  - 当前只统计缺失，不统计重复

## 周期级指标

周期级统计都先按 `送检单编号` 聚合，再看每个周期里整倍体和嵌合胚胎数量。

### 周期无整倍体胚胎率

- 使用列：
  - `送检单编号`
  - `结果解释`
- 计算方式：
  - 周期内 `整倍体胚胎数 == 0` 的周期数 / 总检测周期数

### 周期无整倍体且含有仅嵌合的胚胎率

- 使用列：
  - `送检单编号`
  - `结果解释`
- 计算方式：
  - 周期内 `整倍体胚胎数 == 0`
  - 且 `嵌合异常胚胎数 > 0`
  - 满足上述条件的周期数 / 总检测周期数

### 周期整倍体率

- 使用列：
  - `送检单编号`
  - `结果解释`
- 计算方式：
  - 周期内 `整倍体胚胎数 > 0` 的周期数 / 总检测周期数

### 周期仅 1 个整倍体胚胎率

- 使用列：
  - `送检单编号`
  - `结果解释`
- 计算方式：
  - 周期内 `整倍体胚胎数 == 1` 的周期数 / 总检测周期数

### 周期 ≥2 个整倍体胚胎率

- 使用列：
  - `送检单编号`
  - `结果解释`
- 计算方式：
  - 周期内 `整倍体胚胎数 >= 2` 的周期数 / 总检测周期数
