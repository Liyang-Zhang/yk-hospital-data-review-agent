# V0.2 路由语料回归说明

当前 V0.2 回归集分为两层：

- 轻量路由测试
  - 不依赖真实 workbook
  - 覆盖 normalization、single-metric routing、conflict、follow-up、refuse、route trace
- 数据执行集成测试
  - 依赖当前 PGT-A 快照
  - 只验证医院、时间、年龄筛选和结果结构没有回归

当前建议和 [PGTA问法标准答案基线-20260702.md](/home/zhangly/repo/github/yk-hospital-data-review-agent/docs/PGTA问法标准答案基线-20260702.md) 配合使用：

- 本文档主要回答：`问题应当路由到哪个函数`
- 标准答案基线文档主要回答：`这个函数当前应该返回什么业务结果`

当前路由主干遵循：

- `Question Normalization`
- `Rule-first Intent Parsing`
- `Follow-up Rewrite`
- `Answerability Policy`
- `Function Resolution`

说明：

- 明确单问题优先走本地规则解析，不默认调用 LLM
- 只有模糊、冲突或复杂结构化请求才进入 LLM 兜底场景
- 年龄范围现在是全局 filter，不再只服务于“年龄分层”主题

当前调试建议：

- 开发模式下优先观察 `route_trace`
  - `raw_message`
  - `normalized_message`
  - `filters`
  - `candidate_metric_ids`
  - `resolved_metric_id`
  - `answer_mode`
- 如果出现“答非所问”，先判断是：
  - normalization 不足
  - candidate metric 冲突
  - follow-up 继承过度
  - 还是 data filter 没有真正命中

当前新增的重点稳定性要求：

- `301医院`、`山西妇幼` 这类医院简称要能被规范化
- `25年7月到10月`、`35以上`、`35-37` 这类口语筛选表达要能进入标准 filters
- `送检量和整倍体率` 这类多主题同问必须进入 `clarify`
- `那整倍体率呢 / 按季度看呢 / 那35岁以上呢` 这类追问要先做安全继承，再决定是否执行
- 当追问里已经出现新的明确主题，例如 `再看一下质控情况`，不能继续继承上一轮 metric
- 当追问切换到 `年龄分层` 这种新分析视角时，不能继续继承上一轮的年龄范围筛选

当前题型最少覆盖：

- 明确可答
  - 送检量、整倍体率、年龄分层、质控、结果结构、周期结局、特殊 CNV
- 需要澄清
  - 多主题同问
  - 无法安全继承的追问
- 应拒答
  - 域外问题
  - 跨产品执行
  - 不支持组合

优化时只允许优先调整：

- normalization 规则
- question class 词表
- follow-up inheritance 规则

每次调整后都应回归：

- 同义问法是否仍命中同一 metric family
- 多主题问题是否仍进入 clarify
- 域外问题是否仍拒答
- 带年龄范围的问题是否仍正确传到 filters
- 明显单问题是否仍然不会触发 LLM 兜底
- 明确新主题的追问是否能覆盖旧主题，而不是被上一轮上下文污染

## 建议人工测试问法池

以下问法尽量保持口语化、偏医院老师或临床沟通风格，适合后续人工联调。当前默认都基于 `PGT-A` 单指标主链路。

### 1. 送检量 / 周期规模

- `帮我看一下 2025 年 PGT-A 一共送了多少周期、多少胚胎`
- `我们中心 7 月到 10 月这段时间 PGT-A 送检量大概怎么样`
- `想看一下今年下半年 PGT-A 的送检情况`
- `301 这边 2025 年 PGT-A 周期量和胚胎量分别是多少`（主题：未识别问题，前端默认加载的是山西妇幼的数据，确实应该被拒绝，但需要输出合理的理由）
- `按季度帮我过一下 PGT-A 的送检趋势`（未命中路由，判定：clarify / 当前问题还没有命中可执行的单指标主题。）

预期函数：

- `pgt_total_volume`

### 2. 整倍体率

- `帮我看一下 2025 年 PGT-A 的整倍体率`
- `统计下 2025 年 7 月到 10 月的整倍体率情况`（已对齐基线：路由命中 `pgta_euploid_rate`，时间范围归一化为 `2025年7月到2025年10月`，结果为 `926 / 471 / 50.9%`）
- `我们中心这几个月 PGT-A 整倍体率怎么样`
- `按月看一下 PGT-A 整倍体率变化`
- `今年 PGT-A 整倍体率有没有哪个季度特别低`(命中路由，answer / 问题已通过产品范围、数据支撑和组合有效性校验，可执行“PGT-A 整倍体率”分析。)

预期函数：

- `pgta_euploid_rate`

### 3. 年龄分层 / 年龄筛选

- `按年龄分层看一下 PGT-A 的整倍体率`（已对齐基线：按 `＜35岁 / 35-38岁 / ＞38岁 / 未填写` 四档业务年龄桶展示）
- `想看下高龄患者这边的整倍体率，大于 35 岁这一组`
- `35 到 37 岁这批患者的 PGT-A 结果怎么样`（未命中路由）
- `未填写年龄的这些样本，整倍体率大概多少`
- `41 岁及以上患者这段时间整倍体率情况怎么样`（命中路由，不确定答案是否准确）

预期函数：

- `pgta_euploid_rate`
  - 如果是“按年龄分层”或“年龄范围筛选后的整倍体率”
  - “按年龄分层”应体现为 `breakdown=age`

### 4. 质控 / 检测成功率

- `帮我看一下 2025 年 PGT-A 的质控情况`
- `我们中心这段时间 NA 多不多`
- `想看一下 PGT-A 的检测成功率`
- `35 到 37 岁患者这批样本质控结果怎么样`（命中pgta_quality_overview函数，但是答非所问）
- `按月过一下 PGT-A 的质控波动`(命中路由，但是分析结果有问题)

预期函数：

- `pgta_quality_overview`

### 5. 结果结构 / 嵌合 / 异常率

- `帮我看一下 PGT-A 的结果分布`
- `这段时间 PGT-A 的嵌合率和异常率怎么样`（已对齐基线：2025年7-10月返回整倍体 `471/50.9%`、仅嵌合 `120/13.0%`、异常 `322/34.8%`、质控不合格 `13/1.4%`）
- `想了解一下 PGT-A 异常结构这块的情况`
- `今年 PGT-A 意外发现率大概多少`（已对齐基线：2025年返回意外发现 `120 / 2681 = 4.5%`）
- `异倍体率这边帮我简单汇总一下`

预期函数：

- `pgta_mosaic_abnormal`

### 6. 周期结局

- `帮我看一下 PGT-A 的周期无整倍体率`
- `我们中心周期整倍体结局怎么样`
- `只有 1 个整倍体和大于等于 2 个整倍体的比例分别多少`（命中pgta_euploid_rate，但是答非所问）
- `想看一下周期层面的整倍体结局，不是胚胎层面的`
- `按月看一下周期无整倍体率变化（2025年7月-11月）`（命中路由，能输出比较好的结果）

预期函数：

- `pgta_cycle_indicator_overview`
  - 如用户明确要求“按年龄分层，从周期维度看整体结局”，应体现为 `breakdown=age`

### 7. 特殊 CNV 提示

- `帮我看一下 PGT-A 的特殊 CNV 提示情况`（命中路由，结果准确性不确定）
- `1Mb 到 4Mb 这类综合征相关提示多不多`
- `拟常染色体区域异常这块有多少`（clarify / 当前问题同时落到了多个可执行指标，请先明确你最想看的主指标。）
- `4Mb 到 10Mb 且高比例嵌合这类情况帮我汇总一下`（候选函数：pgta_special_cnv_overview、pgta_mosaic_abnormal；判定：clarify / 当前问题同时落到了多个可执行指标，请先明确你最想看的主指标。）
- `想单独看一下 CNV 提示，不看整体结果结构`

预期函数：

- `pgta_special_cnv_overview`

### 8. 追问链建议

- `先看一下 2025 年 7 月 PGT-A 送检量 -> 那整倍体率呢`
- `先看整倍体率 -> 按季度看呢`
- `先看 7 月到 10 月整倍体率 -> 那 35 岁以上呢`（第一个问题命中路由，第二个未命中，结果准确性未知）
- `先看结果分布 -> 那按季度看呢`
- `先看质控情况 -> 那异常率呢`
- `先看 7 月到 10 月 >35岁整倍体率 -> 换成年龄分层`

预期：

- 能安全继承时进入 `answer`
- 不能安全继承时进入 `clarify`
- `换成年龄分层` 时只继承时间和医院，不继续带入上一轮 `age_range`

### 9. 应澄清 / 应拒答问法

- `帮我分析一下`
- `这个医院情况怎么样`
- `送检量和整倍体率一起帮我汇总一下`（判定：clarify / 当前问题同时落到了多个可执行指标，请先明确你最想看的主指标。）
- `按质控维度看整倍体率`
- `看一下 2025 年 PGT-SR 二阶段结果`（判定：refuse / 当前快照模式已接入多产品文件，但真实执行主链路仍只支持 PGT-A，暂不支持 PGT-SR、PGT-M、PGT-AH 或全产品汇总执行。）
- `高龄、流产、种植失败这些指标帮我一起看下`
- `今天天气怎么样`

预期：

- 缺主指标或多主题同问：
  - `clarify`
- 非法组合、跨产品、域外问题：
  - `refuse`
